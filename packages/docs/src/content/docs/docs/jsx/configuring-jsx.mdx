title: Configuring JSX
description: Configure Lilybird however you would like.
---

import { Tabs, TabItem } from "@astrojs/starlight/components";

To use `@lilybird/jsx` you need to enable `react-jsx`. This can be done in a few ways.

<Tabs>
  <TabItem label="tsconfig.json">
    ```json
    {
      "compilerOptions": {
        "jsx": "react-jsx",
        "jsxImportSource": "@lilybird/jsx"
      }
    }
    ```
</TabItem>
<TabItem label="bunfig.toml">
    ```toml
    jsx = "react-jsx"
    jsxImportSource = "@lilybird/jsx"
    ```
</TabItem>
<TabItem label="jsconfig.json">
    ```json
    {
      "compilerOptions": {
        "jsx": "react-jsx",
        "jsxImportSource": "@lilybird/jsx"
      }
    }
    ```
  </TabItem>
</Tabs>
